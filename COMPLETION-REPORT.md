## 🎨 Personnalisation Facile

### Changer les couleurs
Éditer `--primary-color` dans `css/styles.css`

### Ajouter des produits
Ajouter objets dans `js/products-data.js`

### Modifier le texte
Chercher & remplacer dans les fichiers HTML

### Changer le logo
Remplacer le texte "ShopHub" par une image
